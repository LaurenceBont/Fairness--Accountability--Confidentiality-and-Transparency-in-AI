## Dependencies?
To get all dependencies for this paper use pipenv or conda

### pipenv:
`pip install pipenv`\
`pipenv shell`\
`pipenv sync`\

### conda:
install conda\
`conda env create -f environment.yml`\
`conda activate fact`\

Now al packages needed for this project are installed!
